# portfolio

Персональная монорепа Владимира Беляева (AI Full Stack Engineer · Product Lead · Analytics Engineer) — единая база для личного бренда: сайт, блог-лонгриды и поиск работы. Идея — держать профиль, резюме под разные роли и контент в одном источнике правды, из которого собираются и веб-страницы, и PDF-резюме, и публикации для YouTube / Telegram / лонгридов. Структура: `apps/web` — личный сайт, `docs/cv` — резюме, `DESIGN.md` — дизайн-система сайта (тема «Corona Bloom»), `docs/design-notes.md` — журнал дизайн-решений, `refs/` — банк визуальных референсов. Бренд держится не на логотипе, а на сквозном голосе и паре ритуалов (нумерованные «заметки», единый формат превью, рубрика-«сводка цифр»), связывающих все площадки.
